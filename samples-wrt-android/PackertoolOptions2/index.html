Test Steps

  1. Pack the web app with several options, command like:
    $ python make_apk.py --name=[name] --package=[package] --app-url=http://www.intel.com --arch=x86 --app-version=1.0.0 --keep-screen-on --mode=shared
  2. Pack the web app with several options, command like:
    $ python make_apk.py --name=[name] --package=[package] --app-url=http://www.intel.com --arch=arm --app-version=1.0.0 --keep-screen-on --mode=shared --verbose
  3. Compare the two packaging process messages in terminal
  4. Install the first apk in "x86" platform, and the second one in "arm" platform
  5. Set the device screen off as 15 seconds(Setting->Display->Sleep: 15 seconds)
  6. Check the web app version in Setting
  7. Launch the web app and check the screen

Purpose:

Verifies that Packaging Tool support the 'x86' arch, 'app-url', 'mode=shared', 'verbose' and 'keep-screen-on' options

Expected Result:

  1. Apk with "x86" arch and shared mode could be created successfully
  2. Apk with "arm" arch and shared mode could be created successfully
  3. More detailed messages can be got in the second packaging process
  4. Web App could be installed successfully
  5. Version of the two web apps are both 1.0.0
  6. The screen keep on for more than 15 seconds
  7. Web App could be launched successfully, the home page display intel and keep screen on status more than 15 seconds
