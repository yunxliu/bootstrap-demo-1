## Usecase Design

This sample demonstrates PackertoolOptions2 feature basic functionalities, include:

* Packaging Tool support the "x86/arm" arch, "app-url", "mode=shared", "verbose" and "keep-screen-on" options
to bulid apk package

This usecase covers following method:

* $ python make_apk.py --app-url --mode=shared keep-screen-on --arch=x86/arm --verbose
