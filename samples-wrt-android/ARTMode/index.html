Precondition

  • Make sure the Android version is 4.4 KitKat or highter

Test Steps

  1. Setting Android ART mode enable
    Setting --> Developer options --> Select runtime --> Use ART (rather than 'Dalvik')
  2. Install a Crosswalk App
  3. Launch the web app
  4. Uninstall the web app

Purpose:

Verifies the ART mode can run successfully

Expected Result:

  1. Setting ART mode successfully
  2. Install the web app successfully
  3. Launch the web app successfully
  4. Uninstall the web app successfully
