Test Steps

  1. Pack the web app with command:
    $ python make_apk.py --name=Test --package=org.xwalk.Test --app-url=http://www.baidu.com --app-version=1.0.0
  2. Install the apk with command:
    $ adb install -r Test_1.0.0_x86.apk
  3. Launch the web app
  4. Check the web app version in setting
  5. Pack the web app with command:
    $ python make_apk.py --name=Test --package=org.xwalk.Test --app-url=http://www.baidu.com --app-version=1.0.1
  6. Install the apk with command:
    $ adb install -r Test_1.0.1_x86.apk
  7. Launch the web app
  8. Check the web app version in Setting

Purpose:

Verifies that Crosswalk enable update web app with the different app-version.

Expected Result:

  1. Apk could be created successfully
  2. Web App could be installed successfully
  3. Web App could be launched successfully
  4. Web App version is 1.0.0
  5. Apk could be created correctly
  6. Web App could be updated successfully
  7. Web App could be launched successfully
  8. Web App version is 1.0.1
